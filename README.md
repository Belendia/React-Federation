# React-Federation

## Micro frontend

### create the project

`pnpx create-mf-app`

### set the following for the project

`Pick the name of your app: home
? Project Type: Application
? Port number: 3000
? Framework: react
? Language: javascript
? CSS: Tailwind
? Packer: Webpack`

### install all the necessary packages

`pnpm install`

### start the home app

`pnpm start`
