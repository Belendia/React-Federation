# React-Federation

## Micro frontend

### create the project

`pnpx create-mf-app`

### set the following for the project

Pick the name of your app: home<br>
? Project Type: Application<br>
? Port number: 3000<br>
? Framework: react<br>
? Language: javascript<br>
? CSS: Tailwind<br>
? Packer: Webpack<br>

### install all the necessary packages

`pnpm install`

### start the home app

`pnpm start`
